# Email Automator Developer Documentation

Welcome to the developer documentation for the Email Automator project. This application uses AI to automatically process, categorize, and act on emails from Gmail and Outlook.

## Documentation Index

- **[Setup Guide](./SETUP.md)**: Instructions for installing dependencies, configuring environment variables, setting up the Supabase database, and running the application.
- **[Architecture](./ARCHITECTURE.md)**: Overview of the system architecture, including the backend/frontend split, Supabase reuse strategy, and core module design.

## Quick Start

1.  **Install Dependencies**: `npm install`
2.  **Start Dev Server**: `npm run dev -- --port 3003`
3.  **Start Backend**: `npm run serve -- --port 3002`
4.  **Visit**: http://localhost:3003
