# Email Automator Developer Documentation

Welcome to the developer documentation for the Email Automator project. This application uses AI to automatically process, categorize, and act on emails from Gmail and Outlook.

## Documentation Index

- **[Setup Guide](./SETUP.md)**: Instructions for installing dependencies, configuring environment variables, setting up the Supabase database, and running the application.
- **[Architecture](./ARCHITECTURE.md)**: Overview of the system architecture, including the Async ETL pipeline, local disk storage, and the hybrid Supabase strategy.
- **[Task List](./TODO.md)**: Current development priorities and recently completed milestones.

## Quick Start

1.  **Install Dependencies**: `npm install`
2.  **Start Dev Server**: `npm run dev` (Port 3000)
3.  **Start Backend**: `npm run dev:api` (Port 3001)
4.  **Visit**: http://localhost:3000
